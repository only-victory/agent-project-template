# MEASUREMENTS: 실측 기록 (템플릿이 값을 하는가의 증거)

> "검증 없으면 예쁜 쓰레기"를 템플릿 자신에게 적용하는 파일. 느낌("토큰 많이 쓰는 듯")을 숫자로 바꾼다.
> 기록 시점: /retro 때 한 줄, 또는 session-metrics 훅(활성화 시)이 토큰 열을 자동 기록.
