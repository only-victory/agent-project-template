# PLAN 인덱스: 한눈에 보는 전체 작업

> **파일명 유래**: 모든 PLAN/PRD를 한 장에 모은 `index`(색인). 문서가 100개·200개로 늘면 폴더만 봐선 "뭐가 끝났나"를 못 찾는다. 이 한 장이 전체 상태를 한눈에 보여준다.

> 새 PLAN/PRD를 만들면 여기 한 줄 추가한다. 상태가 바뀌면 이 표의 상태를 갱신한다.
> 완료(Done)되어 활발한 참조가 끝나면 → 문서를 `archive/`로 옮기고, 아래 "완료(보관)" 섹션으로 행을 옮긴다.
> 최종 검토일: 2026-06-25.

## 상태값 정의 (이 표에서 쓰는 말)
| 상태 | 뜻 | 다음 |
|------|----|----|
| `Draft` | 초안: 아이디어·명세 작성 중, 아직 착수 전 | 착수하면 In Progress |
| `In Progress` | 진행 중: 실제로 만들고 있음 | 막히면 Blocked, 끝나면 Done |
| `Blocked` | 막힘: 사람 결정·외부 의존으로 멈춤 | 풀리면 In Progress |
| `Done` | 완료: 검증(골든셋) 통과하고 닫음 | 참조 끝나면 archive로 |

> 검증 판정(Pass/Hold/Retry)은 상태가 아니라 "Done으로 가기 전 게이트 결과"다(`golden-set.md`). 불확실 표시 `[?]`는 로드맵의 미정 항목용(`roadmap.md`).
