# archive: 완료된 PLAN 보관소

> 완료(Done)된 PLAN/PRD 문서를 여기로 옮긴다. 살아있는 작업과 끝난 작업을 눈으로 분리하기 위함.

## 규칙
- **여기로 옮기는 시점**: PLAN이 Done이 되고, 더 이상 수정·참조가 활발하지 않을 때.
- **왜 완료만 옮기나**: 완료 문서는 더 이상 안 바뀌어서 경로가 바뀌어도 링크 깨질 일이 적다. 진행 중 문서는 옮기지 않는다(참조·수정이 살아있어 git/링크가 깨진다).
- **옮길 때**: INDEX.md의 해당 행을 "완료" 섹션으로 옮기고 경로를 `archive/`로 갱신한다. 그 문서를 참조하는 다른 문서(ROADMAP 등)가 있으면 경로를 함께 고친다.
- **옮기기 전 확인**: 이 문서를 아직 활발히 참조하는 곳이 없는지 확인(있으면 보류). 의심되면 옮기지 말고 INDEX 상태만 Done으로 둔다.
