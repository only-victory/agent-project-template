## 1. 비협상 (자주 걸리는 것 요약) · 정본 = @agent-loop.md
> 정본의 복제가 아니라 *매 작업 가장 자주 적용되는 것*만 추린 리마인더. **충돌 시 @agent-loop.md가 이긴다.**

- 명확한 지시엔 **반론 / 되묻기 / 실행** 셋 중 하나. 딴짓·묵묵부답·시키지 않은 일 금지.
- **묻기 전에 먼저 조사**(코드·로그·실데이터·문서). *진행을 막는* 모호함만 질문.
- **증거 없이 "완료" 금지.** §2를 실제로 충족시키고 출력 첨부. 테스트를 통과하게만 고치는 건 검증이 아니다.
- **비가역·외부영향**(삭제·배포·외부 전송·실데이터·force push·마이그레이션)은 `계획 → 승인 → 실행`.
- **작게**: 파일 ~200줄 가이드, 컴포넌트는 단일 책임·가볍게. 더 단순한 길 우선(YAGNI).
- 시크릿을 로그·응답·산출물·커밋에 남기지 않는다.
- 막히면(근거 부족·권한 밖·반복 실패) 멈추고 사람에게.
